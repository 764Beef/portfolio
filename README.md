## What's in here

- **Custom cursor** — smooth trailing ring effect
- **Scroll animations** — reveal-on-scroll for every section
- **Progress bars** — animated learning progress cards
- **Contact form** — hooked up and ready for a backend
- **Responsive** — mobile menu with hamburger toggle
- **No dependencies** — no npm, no bundler, just files

---

## Getting started

```bash
# Clone it
git clone https://github.com/764Beef/portfolio
cd portfolio

# Open it
open index.html

# Or with a local server
npx serve .
# or
python3 -m http.server 3000
```

Open http://localhost:3000 and you're good.

## Project structure

```
├── index.html              # Entry point
├── css/
│   ├── variables.css       # Design tokens (colors, spacing)
│   ├── base.css            # Reset + global styles
│   ├── nav.css             # Navigation + mobile menu
│   ├── hero.css            # Hero section
│   ├── about.css           # About + stat cards
│   ├── skills.css          # Skills grid + pills
│   ├── learning.css        # Learning cards + progress bars
│   ├── projects.css        # Project cards
│   ├── contact.css         # Contact form + quick links
│   ├── animations.css      # Keyframes + scroll reveals
│   └── responsive.css      # Breakpoints
├── js/
│   ├── cursor.js           # Custom cursor logic
│   ├── nav.js              # Mobile menu toggle
│   ├── scroll.js           # Scroll-reveal observer
│   ├── progress.js         # Progress bar animation
│   ├── main.js             # Init + misc
│   └── contact.js          # Form handling
└── img/
    └── photo.jpg           # Profile photo
```
